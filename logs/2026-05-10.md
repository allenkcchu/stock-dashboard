## 遇到的問題與解法

| 問題 | 解法 |
|------|------|
| Railway Crash：`Could not import module "main"` | Root Directory 未設為 `bot`，在 Railway Service Settings 補設 |
| GitHub Actions `LINE_CHANNEL_ACCESS_TOKEN or LINE_USER_ID not set` | Secrets 未在 GitHub repo Settings → Secrets and variables → Actions 建立，補加後解決 |
| LINE push 400：`not allowed to mix different bubble size in a carousel` | header bubble `size: kilo` 與其他 `size: mega` 不能混用，統一改為 `mega` |
| Flex Message 在 LINE 顯示為圖片，無法放大或互動 | 放棄 Flex Message，改為純文字推播；使用者輸入 ticker 觸發新聞分析 |
| `workflow_dispatch` 手動觸發被市場時間檢查擋住 | workflow 加 `inputs.force`，腳本讀 `FORCE_SEND` env var 繞過檢查 |
| workflow 仍跑舊腳本 `send_line_notify.py` | 修正 `daily_notify.yml` 改為執行 `send_line_push.py` 並更新 secrets 名稱 |
